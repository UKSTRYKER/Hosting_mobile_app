Centralize

A mobile academic scheduling application designed to help students manage terms, courses, assessments, reports, and academic progress within a centralized platform.

About the App

Centralize was developed using .NET MAUI and SQLite as part of a software engineering capstone project. The application helps students organize academic information through a streamlined mobile interface with secure authentication, scheduling tools, search functionality, and academic reporting.

Core Features

Secure PIN Authentication

Users authenticate using a secure PIN each time the application launches.

Academic Scheduling

Create and manage academic terms, courses, and assessments within a structured schedule.

Search Functionality

Quickly search courses and assessments using integrated filtering and search tools.

Academic Reports

Generate academic schedule reports, assessment reports, and course progress reports.

SQLite Database

Data is stored locally using SQLite for persistent and reliable academic data management.

Responsive Mobile UI

Built using .NET MAUI with a modern mobile-first user interface and scalable layouts.
